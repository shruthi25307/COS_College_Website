# COS College Website 🎓
LIVE WEBSITE: https://coswebsite.netlify.app

A modern, responsive, and visually appealing college website developed using HTML, CSS, and JavaScript. This project is designed to provide prospective students, parents, and visitors with comprehensive information about the institution, its academic programs, admission process, campus life, and contact details through an intuitive and user-friendly interface.

## ✨ Key Features

### 🏠 Home Page
- Modern hero section with engaging visuals
- College overview and highlights
- Statistics and achievements section
- Featured programs and facilities
- Call-to-action sections for admissions

### 📚 Courses Page
- Detailed showcase of academic programs
- Course categories and specializations
- Program highlights and seat availability
- Interactive course cards with modern design

### 🖼️ Gallery Page
- Dynamic campus gallery
- Categorized image collections
- Interactive image previews and lightbox functionality
- Showcase of infrastructure, events, laboratories, and student activities

### 🎓 Admissions Page
- Admission eligibility information
- Step-by-step admission process
- Online application form
- Frequently Asked Questions (FAQs)
- Academic requirements and guidelines

### 📞 Contact Page
- College contact information
- Inquiry form for prospective students
- Location and communication details
- User-friendly contact interface

## 🎨 Design Features
- Fully responsive layout
- Modern and professional UI/UX
- Smooth animations and transitions
- Dark Mode / Light Mode support
- Elegant typography and color scheme
- Mobile-first optimization

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
